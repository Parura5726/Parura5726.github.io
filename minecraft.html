This is a script to install minecraft on the school computers.
IMPORTANT: First time you run it, close the minecraft launcher as soon as it fully loads, after the language selection dialogue.
IMPORTANT: When you run this installer, log in to minecraft and install instances/mods, do it on a personal hotspot, otherwise this script will break
To run this, download the file below, install Visual Studio Code (from filewave), open it, press ctrl+` (backtick), and type in chmod +x ~/Downloads/minecraft if you're running this for the first time, then, and for any subsequent time you want to run it, ~/Downloads/minecraft.
